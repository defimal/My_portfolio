Key Features

  • Modular structure: Clean separation of logic (data access, business, UI)

  • Secure access: Basic login system with input validation

Technologies Used

Java JDBC SQL MySQL / SQLite Object-Oriented Programming Command Line Interface

Design Philosophy

The app prioritized modularity and maintainability, using OOP principles and design patterns. Each module was version-controlled and delivered iteratively with technical documentation and diagrams.
